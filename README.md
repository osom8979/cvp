Computer Vision Player

## Developing

```shell
## Create a local Python virtual environment.
./python

## Black formatting
./black.sh

## PEP8 linting
./flake8.sh

## Sort import order
./isort.sh

## Type checking
./mypy.sh

## Unit testing
./pytest.sh
```

## Installation

### from pip

```shell
pip install cvp
```

### from source

```shell
git clone https://github.com/osom8979/cvp ~/.cvp.app
cd ~/.cvp.app
./python -V
```
